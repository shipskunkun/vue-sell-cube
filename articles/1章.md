## 1-0 how to add a repository in github

现在github 上建个仓库，然后 git clone 下来  
然后，把本地文件拖进去

``` js
git init
git add README.md
git commit -m "first commit"
git remote add origin https://github.com/shipskunkun/vue-sell-cube.git
git push -u origin master

```
